# Deploy Guide

## Opsi Hosting

- Netlify.
- Vercel.
- GitHub Pages.
- Cloudflare Pages.

## Langkah Cepat

1. Pastikan semua aset gambar sudah ada di `assets/images/`.
2. Uji lokal dengan server statis.
3. Upload folder project ke hosting pilihan.
4. Pastikan domain mengarah ke root project.
5. Cek kembali `robots.txt`, `sitemap.xml`, canonical, dan OG tags.

## Indexing Google

- Submit `sitemap.xml` ke Google Search Console.
- Inspect URL homepage.
- Pastikan title, description, dan struktur heading konsisten.
- Gunakan gambar OG untuk share sosial.

## Checklist Produksi

- Gambar tidak broken.
- Favicon tampil.
- Mobile menu berfungsi.
- Slider testimoni jalan.
- Lighthouse SEO dan Accessibility layak.
