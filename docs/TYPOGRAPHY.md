# Typography Guideline

## Font

- Heading: Poppins 600/700/800.
- Body: Inter 400/500/600/700.

## Hierarki

- H1 hanya satu, di hero.
- H2 untuk judul section.
- H3 untuk card title dan judul footer.

## Ukuran dan Ritme

- Body minimum 16px.
- Hero title besar dan padat.
- Gunakan line-height longgar untuk deskripsi agar mudah dibaca.

## Prinsip

- Jangan memakai terlalu banyak weight dalam satu tampilan.
- Jaga kontras tipografi di atas latar gelap.
- Spacing antar blok harus terasa mewah, bukan rapat.
